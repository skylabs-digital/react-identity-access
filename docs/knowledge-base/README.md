# Knowledge Base

Accumulated functional documentation for the `react-identity-access` library.

## Contents

This directory stores business rules, architectural decisions, and domain knowledge that evolves with the project. Each completed feature should have its key learnings documented here.

## Structure

- **Business rules** documented per feature
- **Scenarios** (happy paths, edge cases, errors)
- **API contracts** and expected behaviors
- **Migration notes** for breaking changes
